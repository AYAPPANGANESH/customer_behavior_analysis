## How to Run

1. Clone or download this repository.
2. Install the required Python libraries:

   ```bash
   pip install pandas numpy matplotlib seaborn
   ```
3. Open the Python notebooks and execute the cells in order.
4. Import the cleaned dataset into PostgreSQL, MySQL, or SQL Server.
5. Run the SQL scripts provided in the `SQL` folder.
6. Open the Power BI dashboard (`.pbix`) to explore the visualizations.
7. Review the project report and presentation for detailed insights.
